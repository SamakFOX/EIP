### 개요
정보처리기사는 [한국산업인력공단](https://www.hrdkorea.or.kr)이 시행하고 [Q-Net](https://www.q-net.or.kr/crf005.do?id=crf00503&jmCd=1320)을 통해 매년 3회 정기시험을 진행하는 과목입니다.  

| | 필기 | 실기 |
|:---:|:---:|:---:|
| 과목 | 1. 소프트웨어설계<br>2. 소프트웨어개발<br>3. 데이터베이스구축<br>4. 프로그래밍언어활용<br>5. 정보시스템구축관리 | 전과목 |
| 문항유형 | 객관식 (4지선다) | 주관식 (필답형) |
| 문항수 | 총 100문항<br>과목당 20문항<br>문항 당 1점 | 총 20문항<br>문항 당 5점 |
| 커트라인 | 전체 60점(60문항)<br>과목당 40점(8문항) | 전체 60점(12문항) |

시험 정보는 위와 같으며, 검정 수수료는 필기: 19,400원 / 실기: 22,600원입니다.  


아래 12가지 분야를 얼마나 알고있는지 확인하는 시험이며  
전공자라면 전공 과목에서 배우는 내용이 많이 나오므로 조금 더 쉽게 이해가 될 수 있습니다.  
> 01 - 현행 시스템 분석 및 요구사항 확인 (소프트웨어 공학 기술의 요구사항 분석 기법 활용)  
> 02 - 데이터 입출력 구현 (논리, 물리데이터베이스 설계, 조작 프로시저 등)  
> 03 - 통합 구현 (소프트웨어와 연계 대상 모듈간의 특성 및 연계 모듈 구현 등)  
> 04 - 서버프로그램 구현 (소프트웨어 개발 환경 구축, 형상 관리, 공통 모듈, 테스트 수행 등)  
> 05 - 인터페이스 구현 (소프트웨어 공학 지식, 소프트웨어 인터페이스 설계, 기능 구현, 구현검증 등)  
> 06 - 화면설계 (UI 요구사항 및 설계, 표준 프로토 타입 제작 등)  
> 07 - 애플리케이션 테스트 (테스트 케이스 설계, 통합 테스트, 성능 개선 등)  
> 08 - SQL 응용 (SQL 작성 등)  
> 09 - 소프트웨어 개발 보안 구축 (SW 개발 보안 설계, SW개발 보안 구현 등)  
> 10 - 프로그래밍 언어활용 (기본 문법 등)  
> 11 - 응용 SW기초 기술 활용 (운영체제, 데이터베이스 활용, 네트워크 활용, 개발환경 구축 등)  
> 12 - 제품 소프트웨어 패키징 (제품 소프트웨어 패키징, 제품소프트웨어 매뉴얼 작성 , 버전 관리등 )
